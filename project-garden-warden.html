Garden Warden

Immersive Simulation focused on ecological restoration. UE5 C++ — In-world inspection shaders, physics-based interaction, and scene-capture UI.

Work In Progress

This project is currently in active development. The mechanics below are fully implemented in C++.

Garden Warden is a simulation-adventure game built in Unreal Engine 5 using C++. You play as a hero with gardening powers who washes ashore on a mysterious island, helping distinct communities resolve their problems through ecological restoration. The game prioritizes immersive in-world mechanics over traditional UI, immersing the player in the physical act of gardening.

Technical Implementation

  • In-World Inspection System (C++ & Shaders):

    Instead of UI meters, plant needs are hidden. I engineered a "Lens Mechanic" that allows players to inspect plants in 3D. This system dynamically manipulates Material Instances to render specific stress signals—browning leaves, cracked planters, dry soil—that are only visible through the lens viewport.

  • Physics-Based Planting Mechanic:

    Planting is not a single click. I implemented a physics-based minigame where players adjust pitch, yaw, and launch power to land seeds in specific planter openings. The system calculates trajectories in real-time and accounts for biome-specific physical materials (e.g., sticky mud, bouncy fungal floors, sliding gravel).

  • Real-Time 3D Dialogue System:

    Developed a custom UI pipeline using Scene Capture 2D components. Instead of static 2D avatars, the dialogue system captures the actual 3D NPC—preserving their current animation state and lighting—and projects it onto the 2D dialogue interface for seamless immersion.
